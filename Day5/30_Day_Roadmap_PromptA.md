# 30-Day Learning Roadmap: Data Science for Beginners

*A practical, beginner-friendly plan — no prior experience assumed.*

---

## 🎯 Final Outcome

By Day 30, you will be able to:
- Write Python scripts to clean, analyze, and visualize data
- Understand core statistics used in data science
- Build and evaluate a basic machine learning model
- Complete and showcase 2–3 small projects in a portfolio (e.g., GitHub)

## 💼 Projects

1. **Data Cleaning Project (Week 2)** — Take a messy public dataset (e.g., Titanic or a retail sales CSV) and clean it: handle missing values, fix types, remove duplicates.
2. **EDA Report (Week 3)** — Pick a dataset (e.g., movies, sports, or e-commerce data) and produce a short exploratory analysis with 4–5 visual insights.
3. **End-to-End ML Mini-Project (Week 4)** — Predict an outcome (e.g., house prices or customer churn) using a simple model, evaluate it, and document your process in a GitHub README.
